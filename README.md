# 프로젝트명: 연애판별소(연판소)

## 1. 개요
- **개요**: 썸 단계에서 발생하는 관계의 불확실성을 사주로 분석하여, '썸 손절 판별 리포트'를 제공하여, 사용자가 더 빠르고 현명한 연애 의사결정을 할 수 있도록 지원한다.
- **목적**: 애매한 썸을 확실하게 판별하여, 시간과 감정을 낭비하지 않도록 돕는 서비스
- **주요 타겟**: 썸 단계에서 관계의 불확실성으로 고민하는 **20대 여성**
- **핵심 가치**: 애매한 관계를 끝내고, 확신 있는 선택을 돕는 서비스
- **핵심 기능**: 사주 분석을 기반으로 관계의 지속 여부를 판단하는 **‘썸 손절 판별 리포트’** 제공
- **주요 콘텐츠**: 상대의 연애 성향, 관계 흐름, 발전 가능성, 위험 신호, 행동 가이드

## 2. 문제 (Problem)
20대 여성들은 썸 단계에서 가장 큰 스트레스를 받는다.
왜냐하면 아래 이유들 때문이다.
- 상대가 나를 좋아하는지 모르겠다
- 계속 가도 되는 관계인지 모르겠다
- 시간을 더 써야 하는지 모르겠다
- 손절해야 하는 타이밍을 모르겠다

특히 **"애매한 관계"** 가 가장 큰 문제다.
사귀는 것도 아니고, 끝난 것도 아니고, 기다려야 할지 포기해야 할지 모르는 상태.
결국
- 감정 소모
- 시간 낭비
- 자존감 하락
으로 이어진다.

## 3. 해결 (Solution)
연판소는 썸 상대와의 관계를 분석하여
👉 **"계속 가도 되는 관계인지"**
👉 **"손절하는 것이 나은 관계인지"**
를 판별해주는 서비스다.

리포트에는
- 상대의 연애 성향
- 현재 관계 흐름
- 관계 발전 가능성
- 위험 신호
- 행동 가이드
를 포함하여 사용자가 더 이상 혼자 고민하지 않도록 돕는다.

## 4. 기대효과 (Benefit)
1) 감정 낭비 감소 : 애매한 관계에 더 이상 시간을 쓰지 않는다.
2) 빠른 의사결정 : 계속 갈지, 정리할지, 판단할 수 있다.
3) 자존감 보호 : 상대의 행동에 의미 부여하며 혼자 상처받는 시간을 줄인다.
4) 관계 이해도 향상 : 상대의 행동과 심리를 더 객관적으로 이해할 수 있다.
5) 미래 리스크 예방 : 몇 개월 뒤 후회할 관계를 미리 걸러낼 수 있다.
